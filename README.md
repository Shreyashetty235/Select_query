# Select_query

This task is to practice writing basic SQL SELECT queries to extract data from one or more tables.
This includes:
Using SELECT * and specific columns names to extract required data
Applying WHERE, AND, OR, LIKE, BETWEEN
Sorting results using ORDER BY

Database Table Used:
Employees table
Department table

Tool used:
MYSQL Workbench

Conculsion:
Through this task, a clear understanding of SQL data retrieval operations and how to retrieve  data from one or more tables.
